Semana 2

El martes 27 de agosto el profesor dio una explicación a las definiciones de HTML, que significa HyperText Markup Language (Lenguaje de Marcado de Hipertexto), es el lenguaje básico que se utiliza para crear páginas web. Es la estructura o esqueleto de un sitio web. HTML organiza el contenido, como textos, imágenes, videos, y enlaces, en un formato que los navegadores web pueden entender y mostrar de manera ordenada para que los usuarios puedan ver y navegar por la página. .

Ese día también, realizamos el cambio de imagenes en la bitácora utilizando el Visual Studio Code con los siguientes pasos:

-Primero: Busca la imagen y cambiale de nombre. Dentro del archivo HTML, encuentra la línea de código que tiene la etiqueta "". Esta etiqueta es la que muestra la imagen en tu página.

-Luego:Cambia la ruta de la imagen: Sustituye el valor dentro del atributo src="" por la ruta de la nueva imagen que quieres usar. Por ejemplo, cambia src="imagenes/vieja.jpg" a src="imagenes/nueva.jpg".

-Guarda los cambios: Haz clic en "Archivo" y luego en "Guardar", o simplemente presiona Ctrl + S.

-Visualiza el cambio: Abre tu página web en un navegador para ver cómo se ve la nueva imagen.
